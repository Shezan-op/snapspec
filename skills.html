Extending AI Capabilities

Systematic Agentic Skills

Skills are reusable, isolated packages of knowledge designed to extend what AI coding agents can do in your workspace.

Core Pillars

Instructions

Step-by-step guidance defining how to analyze, scaffold, and implement specific UI and software design models.

Best Practices

Rigid conventions, styling tokens, error recovery pipelines, and standards preventing common AI coding mistakes.

Scripts & Docs

Automated build helpers, validation checks, and templates allowing agents to test code before final delivery.

Installation Examples

Install a specific skill from a repository:

npx skills add https://github.com/eddiebe147/claude-settings --skill 'Course Creator'

Install an entire folder of curated skills:

npx skills add bergside/awesome-design-skills

Featured Library

a11y-audit

Performs WCAG accessibility standard checks, scans HTML layouts, and flags color contrast failures.

privacy-pii-redactor

Locates and redacts personally identifiable information (PII) from logs and environment configurations.

Applies Awwwards-tier UI patterns, nested bezels, fluid transitions, and premium OLED black texture rules.

security-reviewer

Scans codebase inputs for SSRF, injection vectors, hardcoded secrets, and general OWASP compliance.
