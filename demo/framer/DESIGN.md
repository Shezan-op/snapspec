# Overview

This homepage is a long-scroll, black-canvas marketing site.

The visual system prioritizes:

1. Large typography before imagery.
2. Product credibility through interface screenshots.
3. Community and ecosystem content.
4. Dense visual showcases.
5. High contrast minimalism.

The page is not driven by gradients, decorative illustrations, or abstract branding graphics.

Instead, the homepage communicates value through:

- Product UI screenshots
- Creator content
- Marketplace assets
- Community proof
- Large editorial-style headlines

The entire experience sits on a near-black background.

Most sections use large amounts of vertical spacing rather than visible dividers.

Content blocks appear as independent modules floating on a shared black canvas.

Visual hierarchy is created through:

- Scale
- Contrast
- Spacing
- Screenshot density

rather than through colorful backgrounds or strong borders.

# Do / Don't

## Do

- Keep black canvas dominant.
- Preserve large typography hierarchy.
- Use real visual content heavily.
- Maintain generous spacing.
- Use rounded geometry consistently.
- Keep screenshots as primary storytelling device.

## Don't

- Introduce colorful section backgrounds.
- Add aggressive gradients.
- Use heavy drop shadows.
- Replace screenshots with illustrations.
- Reduce whitespace significantly.
- Use bright accent colors as dominant UI surfaces.
