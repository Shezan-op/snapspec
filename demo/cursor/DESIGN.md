# Overview

This homepage is a long-scroll SaaS marketing website built around a single dominant visual principle:

**Large product demonstrations surrounded by minimal marketing chrome.**

The design uses a very dark canvas throughout nearly the entire experience. Most sections alternate between:

- Product screenshot + explanation
- Explanation + product screenshot
- Customer proof
- Use cases
- Educational content
- Final conversion section

The interface avoids bright gradients, aggressive illustrations, decorative graphics, floating shapes, excessive animations, or trendy AI aesthetics.

Instead, the visual identity is driven by:

- Product UI screenshots
- Large dark surfaces
- Sparse accent color usage
- White typography
- Structured spacing
- Editorial-like content blocks

The product remains the hero throughout the entire page.

# Do

- Keep dark-first visual system
- Prioritize product screenshots
- Use restrained accent color
- Maintain large spacing
- Use simple typography
- Keep screenshots larger than marketing graphics
- Preserve alternating section rhythm

---

# Don't

- Add gradients everywhere
- Add glowing AI effects
- Add glassmorphism
- Add excessive color accents
- Replace screenshots with illustrations
- Use decorative visual noise
- Use oversized shadows
- Introduce highly saturated backgrounds
