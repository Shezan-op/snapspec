# Overview

This homepage is a vertically stacked marketing experience built around oversized food imagery, highly compressed display typography, bold red color usage, and hand-crafted collage aesthetics.

The design does not attempt to feel premium, minimalist, or technology-focused.

Instead, the visual system is loud, playful, energetic, and intentionally exaggerated.

The page combines:

- Oversized headline typography
- Cutout food photography
- Sticker-like illustrations
- Wavy section transitions
- Hand-drawn decorative elements
- Large color-blocked content bands
- Repeated food-related visual motifs

The homepage is primarily composed of five large visual chapters:

1. Hero section
2. Red experience section
3. Ingredient/quality section
4. Travel/community section
5. Lifestyle/change section
6. Large closing wordmark section

The dominant visual principle is:

**Large typography + food photography + playful illustrations + irregular organic section shapes**

The page intentionally avoids rigid corporate grids.

Instead it embraces visual imbalance while maintaining a clear vertical reading flow.

# Do / Don't

## Do

- Use oversized typography.
- Keep food imagery dominant.
- Maintain strong red contrast.
- Use organic section transitions.
- Allow overlap between text and imagery.
- Use playful decorative graphics.
- Preserve collage feeling.

## Don't

- Introduce corporate UI patterns.
- Use rigid grid-heavy layouts.
- Add heavy interface chrome.
- Replace cutout imagery with stock layouts.
- Remove visual overlaps.
- Introduce minimal monochrome styling.
- Flatten section transitions.

---

# Iteration Guide
