# Overview

This homepage is a vertically scrolling luxury product experience built almost entirely on a black canvas.

The dominant visual rule is simple:

**Dark background + premium photography + minimal interface chrome + centered typography.**

The interface avoids decorative UI elements and instead allows product imagery to carry most of the emotional weight.

Large black gaps separate major sections and create breathing room. Typography is generally white, lightweight, and restrained. Product cards, portrait cards, and editorial imagery act as the primary content containers.

The overall structure feels more like a luxury catalog than a traditional ecommerce homepage.

The page consists of:

1. Minimal navigation header
2. Hero banner
3. Featured product release section
4. Collection carousel/grid
5. Editorial storytelling section
6. Friends & Partners section
7. Large craftsmanship showcase section
8. Extended negative-space region
9. Feature/advantage cards
10. Media gallery section
11. Press logos section
12. Footer

No visible bright accent color dominates the experience. Any accent usage is extremely limited and appears only in small action links.

# Key Characteristics

- Predominantly black background across entire page
- Heavy use of luxury watch photography
- Minimal navigation
- Large vertical spacing between major content groups
- White typography on dark surfaces
- Product imagery receives visual priority over copy
- Rounded card corners throughout
- Consistent card-based layout system
- Strong use of cinematic shadows
- Editorial-style storytelling blocks
- Large empty space intentionally used as a design tool
- Premium retail aesthetic rather than SaaS aesthetic
- Small secondary action links using a subtle accent color
- Sections separated primarily through spacing rather than borders

# Do / Don't

## Do

- Use black as the dominant canvas
- Prioritize photography over interface decoration
- Maintain large spacing between sections
- Keep typography restrained
- Use white text predominantly
- Preserve rounded card language
- Use accent color sparingly

## Don't

- Introduce bright UI gradients
- Add colorful interface chrome
- Use heavy glassmorphism
- Add large shadows
- Overcrowd sections
- Replace photography with illustration-heavy layouts
- Use dense copy blocks
