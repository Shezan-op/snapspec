FAQ

          
          

            
            

              

                

                  
Who can use this tool?

                  

                    
                  

                

                

                  
This tool is designed for developers, UI/UX designers, product managers, and founders who want to rapidly translate visual designs or competitor websites into structured, actionable design tokens and documentation for AI-assisted coding.

                  

                    
                  

                

              

            


            
            

              

                

                  
What is the benefit of DESIGN.md files?

                  

                    
                  

                

                

                  
A DESIGN.md acts as a structured prompt or context file for AI coding assistants (like Cursor, GitHub Copilot, or Claude). It ensures that the AI adheres strictly to your brand's spacing, colors, typography, and styling rules, resulting in fewer hallucinations and perfect UI consistency.

                  

                    
                  

                

              

            


            
            

              

                

                  
Does this tool store my data?

                  

                    
                  

                

                

                  
We respect your privacy. Everything is processed directly using your configured API keys (or our free tier endpoints). We do not permanently store your uploaded images or generated markdown files on our servers. Your API keys are saved securely and locally in your browser.

                  

                    
                  

                

              

            


            
            

              

                

                  
What can this tool not do?

                  

                    
                  

                

                

                  
This tool is meant for design system extraction, not full code generation. It will not give you a complete working React or HTML website directly from an image. It produces the architectural blueprint (the DESIGN.md), which you then feed into your AI IDE to generate the actual code.

                  

                    
                  

                

              

            


            
            

              

                

                  
Which AI models are recommended for generation?

                  

                    
                  

                

                

                  
We recommend using vision-capable frontier models like Claude 3.7 Sonnet or GPT-4o for complex layouts, and Gemini 2.5 Flash for fast, cost-effective design system extractions.

                  

                    
                  

                

              

            


            
            

              

                

                  
Can I use local models with this creator?

                  

                    
                  

                

                

                  
Yes! You can connect to a local Ollama instance by entering your local endpoint URL (e.g., http://localhost:11434) and typing the model name in settings.
