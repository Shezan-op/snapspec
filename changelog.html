Changelog

v1.1.1 — Error Resilience & Layout Optimization

June 14, 2026

  • Default Ollama Configuration: Set Ollama as the default provider and pre-populated the API key with the user's test key for a seamless out-of-the-box experience.
  • Zero-loss Error Handling: Generation errors no longer reset the application or clear the uploaded screenshot, allowing settings changes and quick retries.
  • Inline Error State Panel: Failed generations display error details directly in the results card via a new #markdown-output-error view, featuring a primary "Retry Generation" action.
  • Footer Bottom Spacing Reduction: Adjusted the layout wrapper and footer padding-bottom classes to eliminate the large empty vertical gap at the bottom of the homepage.

v1.1.0 — UI, UX, & Settings Upgrade

June 13, 2026

  • Natural Loading Statuses: Added organic, randomized rotating text status updates while generating specifications (no fake progress bars/percentages).
  • Custom LLM Model Support: Enabled a toggle in API settings to input custom model names for full choice flexibility.
  • Show More Demos: Redesigned the demo section to show only 5 default items, keeping the above-the-fold layout clean.
  • Distinct Demo Actions: Visualized the DESIGN.md link as a file badge and the live preview as a prominent primary action button.
  • Skills Page Implementation: Redesigned skills.html as a true, valuable product specifications overview featuring copyable install commands.
  • Integrated Footers & Tighter Spacing: Removed harsh boxed outline borders on the main footer, simplified secondary pages footer structure, and trimmed blank spacing at the bottom of the page.
  • Instant Output Panel Visibility: Toggled the output panel to display skeletons and status texts immediately upon clicking generate, scrolling down to it smoothly.
  • Preset Models Refresh: Added Gemini 2.5 Pro, Claude 3.7 Sonnet, and GPT-4o Mini presets.
  • Mobile Preview Auditing: Certified viewport configurations on all preview HTML files for responsive, standalone PWA layouts.
  • Ollama Cloud Models Routing: Dynamically route Ollama API calls to the remote proxy /api/ollama-cloud when an API key is present, resolving the "failed to fetch" CORS issue.
  • Ollama Cloud Presets Locking: Replaced the outdated Ollama models list with the exact cloud models supported: qwen3-vl:235b-cloud, deepseek-v3.1:671b-cloud, gpt-oss:20b-cloud, gpt-oss:120b-cloud, kimi-k2:1t-cloud, qwen3-coder:480b-cloud, and glm-4.6:cloud.

v1.0.0 — Initial Setup & Enhancements

June 13, 2026

  • Replaced static wordmark image with crisp, scalable live text (Inter font, bold) for improved performance and retina-ready display.
  • Added a "Frequently Asked Questions" (FAQ) section using a clean dropdown/accordion style.
  • Implemented a new "Subscribe" popup modal to capture interest when users copy or download DESIGN.md blueprints.
  • Added a "Show More" functionality to the DESIGN.md markdown preview container to prevent excessive scrolling.
  • Redesigned the footer layout to align with premium design aesthetics.
  • Updated all demo preview links to open in a new tab (target="_blank") to preserve user workflow.
  • Created the initial skills.html and changelog.html skeletons.
